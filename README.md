# Xuhao Hu Academic Homepage

Static academic homepage for GitHub Pages.

## Local Preview

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## GitHub Pages

Use GitHub Pages with:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/ (root)`

The site is intended for a `hxhcreate.github.io`-style repository or any repository served by GitHub Pages.
